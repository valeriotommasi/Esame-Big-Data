# Progetto Esame Big Data

Repository realizzata per l'esame di Big Data.

Il progetto contiene tre esempi di database:

1. Database relazionale SQL - Agenzia viaggi
2. Database NoSQL a grafo - Neo4j - Mobilità urbana
3. Database NoSQL documentale - Elasticsearch - Traccia libera

## Struttura della repository

- `database_relazionale_agenzia_viaggi/`: database SQLite per la gestione di un'agenzia viaggi.
- `database_grafo_neo4j_mobilita/`: database a grafo Neo4j per una rete di mobilità urbana.
- `database_documentale_elasticsearch/`: database documentale Elasticsearch.

## Tecnologie utilizzate

- SQLite
- Python
- Neo4j
- Elasticsearch
- GitHub
